## ✨ Key Features

- ✅ **Unlimited Radio Stations** - Add unlimited stations with stream URLs, images, categories
- ✅ **Beautiful Modern Player** - Audio visualizer, volume control, status indicators
- ✅ **35 Embeddable Widget Styles** - Header, Card, Box designs with custom colors
- ✅ **Radio Browser Import** - Import from 30,000+ worldwide stations
- ✅ **Auto SEO** - Meta descriptions, keywords, Open Graph tags
- ✅ **XML Sitemap** - Auto-generated SEO sitemaps
- ✅ **Country & Language Filter** - Filter stations by country, language, category
- ✅ **Dark Mode** - Built-in dark/light mode toggle
- ✅ **Fully Responsive** - Perfect on desktop, tablet, and mobile
- ✅ **Elementor Support** - 4 custom Elementor widgets included
- ✅ **Submit Radio Form** - Allow users to submit stations with approval workflow
- ✅ **Real Listener Count** - Live stats from SHOUTcast/Icecast servers
- ✅ **Social Sharing** - Share on Facebook, Twitter, WhatsApp, Telegram
- ✅ **Import/Export** - Bulk import via CSV/JSON files
- ✅ **Pages Manager** - Edit About, Privacy, Contact pages from admin
