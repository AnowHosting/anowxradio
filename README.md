```

[anowxradio_player id="123"]                   - Display single radio player
[anowxradio_search]                            - Display AJAX search bar
[anowxradio_categories]                        - Display category list
[anowxradio_submit_radio]                      - Display Submit Radio form
[anowxradio_contact]                           - Display Contact form
```

## 📋 Installation

### Automatic Installation

1. Go to **Plugins → Add New** in WordPress admin
2. Upload the plugin ZIP file
3. Click **Install Now** and then **Activate**
4. Go to **Settings → Permalinks** and click **Save Changes**
5. Plugin automatically creates all necessary pages
6. Start adding radio stations!

### After Activation

1. Navigate to **AnowXRadio** menu in admin
2. Go to **Settings** to configure options
3. Set **Default Country** for homepage filtering
4. Go to **Appearance** to customize colors
5. Visit `yoursite.com/radio-widgets/` to create embeddable players
